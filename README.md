# CG

## Compiling (linux)
```bash
./build.sh
```

## Running
### Generator
```bash
target/generator/generator FILENAME OPTION [PARAMS...]
```

### Engine
```bash
target/engine/engine [SCENE]
```
If no scene is provided `config.xml` will be used.

### Help
Both programs support a `--help` flag to provide some guidance on how to use them.
